# Smart Ambulance Assignment

This project analyzes patient vitals and detects risk level.

Steps done:
1. Data cleaning using Python (Colab)
2. Anomaly detection
3. Risk classification
4. Visualization plots
5. Flask app to show results

Files included:
- vitals.csv
- vitals_clean.csv
- anomaly.csv
- risk.csv
- final_output.csv
- spo2_plot.png
- anomaly_plot.png
- app.py
- requirements.txt

Run project:

pip install -r requirements.txt
python app.py
